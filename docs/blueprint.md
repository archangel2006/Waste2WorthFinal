## Style Guidelines:

- Primary color: Earthy green (#75A478) to symbolize sustainability and nature.
- Background color: Very light desaturated green (#F0F4F0).
- Accent color: Warm orange (#D9882C) to represent the energy and warmth of community collaboration.
- Body font: 'PT Sans' a modern sans-serif which has personality, well-suited as a body font.
- Headline font: 'Playfair' which gives an elegant and fashionable feel.
- Use clean, minimalist icons to represent food types, storage methods, and user roles.
- Maintain a card-based layout for food listings, making it easy to browse donations.
